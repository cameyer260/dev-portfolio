# Example Image Assets

These are the image files currently expected by the examples pages.

If a file is already present, keep the same filename and replace it in place when upgrading the photography later.

Current pages require two images per industry:

- `hero.jpg`
- `proof.jpg`

Put them here:

- `public/examples/{slug}/hero.jpg`
- `public/examples/{slug}/proof.jpg`

## What Each Image Should Be

`hero.jpg`

- The main above-the-fold image
- Should feel like the strongest first impression for that business

`proof.jpg`

- A supporting image used in the trust/proof section
- Should feel more specific and believable than the hero image
